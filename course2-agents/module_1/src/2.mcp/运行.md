## 运行

如何运行

在项目根目录执行（或进入该目录后）：

#### 1 直接跑 client（它会自己以子进程启动 server）：
```sh
cd course2-agents/module_1/src/2.mcp
python client.py
```
输入例如：帮我查一下今天上海的天气，并解释一下。

#### 2 Server / client 逻辑划分：
```py
server.py = MCP server（工具端，只关心“实现 get_weather / search_info_by_tavily”）。
client.py = MCP client（Agent 端，负责调大模型、决定是否用工具、通过 MCP 调用工具）。
```
