## 回归与分类
回归（Regression）与分类（Classification）是监督学习（Supervised Learning）中最基本的两大任务类别。它们的区别主要在于输出的类型：

### 1 基本概念
#### 回归（Regression）
- 目标：预测一个连续的数值
- 输出类型：实数（浮点数或整数），可以是任何数值
- 常见用途：预测价格、温度、时间、长度、概率值等

#### 分类（Classification）
- 目标：预测一个离散的类别标签
- 输出类型：分类标签（如 A/B/C 或 0/1），类别数量有限
- 常见用途：识别物体类别、判断是否有病、情感倾向（好/坏）等

### 2 常见的任务
下面我们用你提到的几个任务来区分哪些是回归，哪些是分类：

| 任务                                    | 回归 or 分类          | 理由                                                            |
| ------------------------------------- | ----------------- | ------------------------------------------------------------- |
| 🎯 目标检测（Object Detection）             | **同时包含回归和分类**     | - **分类**：识别物体是什么（人/车/猫）<br>- **回归**：预测边界框的位置（x, y, w, h 是连续值） |
| 📹 目标跟踪（Object Tracking）              | **主要是回归**（也可能含分类） | - 通常连续预测物体的位置和运动轨迹（坐标）<br>- 有些跟踪器也会重新识别目标类别（分类）               |
| 🏠 房价预测                               | **回归**            | - 输出是一个具体的价格（如123456元），是连续值                                   |
| 🔍 NER 实体识别（Named Entity Recognition） | **分类**            | - 对文本中每个词分类（如人名/地点/组织/无实体），属于标签分类问题                           |
| 📄 文本分类（Text Classification）          | **分类**            | - 把一段文本归入某个类别，如体育/娱乐/科技，或积极/消极                                |
