## 大模型训练、微调、部署
3 天，9:00-12:00，14:00-17:00
### day1 大模型训练与微调基础
- 理论：模型训练流程、迁移学习与 LoRA 微调原理
- 案例：基于开源模型（如 LLaMA-7B、Qwen-7B）微调模型
- 实操：使用 Hugging Face、modelscope 完成模型微调配置
- 无 Python 基础：使用可视化平台/低代码工具，理解和体验模型训练、迁移学习和微调过程
- 有 Python 基础：基于 LlamaFactory 等工具，手动编写和调试微调脚本，实现个性化训练

### day2 模型部署与性能评估
- 理论：模型推理显存占用、评估指标设计
- 案例：将微调模型部署至本地服务器，对比原始模型效果，设计 benchmark
- 实操：使用 VLLM 加速推理，监控 GPU 利用率
- 需要Python 基础：编写脚本实现模型推理、效果评测及性能监控，灵活调整评估流程

### day3 综合实践：全流程案例实战

- 案例：从数据处理到线上部署完整链路
- 数据阶段：清洗业务数据，构建训练数据集，构建分类语料库
- 训练阶段：微调开源模型并评估准确率
- 部署阶段：通过 API 接口提供服务，配置流量监控
- 需要Python 基础：全程使用 Python 实现数据处理、模型微调、部署上线及效果验证
