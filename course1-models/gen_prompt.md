以下为我们华泰证券 "AI涨乐APP" 业务部门定义的完整意图列表（供模型参考，不需在输出中复述）：
```json
{INTENT_JSON}
```
你是一名高质量 NLU 训练数据生成器，任务是为意图分类模型生成用户真实表达。

请根据 意图名称：{{INTENT_NAME}} 生成 {{N}} 条自然语言查询。

## 生成要求（必须全部满足）
### 1. 语言风格多样化
每条表达需随机呈现不同风格，包括但不限于：
- 口语 / 书面语
- 非专业 / 专业术语
- 省略句、不完整表达
- 模糊但偏向该意图的说法
- 请求、抱怨、疑惑、命令等不同语气
- 同义词改写、换一种问法

### 2. 内容限制
- 必须能真实代表该意图
- 不得与其他意图混淆
- 不得包含模型提示、身份描述（如“我是AI”）
- 不得出现模板化、机械重复结构
- 需像真实用户表达：自然、有差异、偶尔口误

### 3. 意图单一性
每条表达只能对应该意图，不得混入其他任务描述

### 4. 输出格式（严格遵守 JSONL，一行一条）
格式如下：
```json
{"query": "用户表达", "intent": {INTENT_NAME}, "label": {{INTENT_LABEL}}}
```
不要输出额外文字，不要输出代码块外内容。

如果你理解，请直接开始生成 {{N}} 条符合要求的表达。
